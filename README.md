# 🤐 Password-Validator
A multi-criteria password validation system, featuring checks for 7 safety traits. Generation of valid strong passwords containing all required is also supported.
 **Codebase is private. Please reach out if you'd like access**.

## 🧠 Project Overview

This Password Validator has the following components:
- Length check
- Capital check
- Number check
- Symbol check
- Specific content check
- Digit sum check
- Palindrome check
- Strong password generator 

## 🔧 Technologies & Tools

C, Bash script

## 🏃:How to use it

Input commands to check for specific requirement, or generate strong password satisfying all the checks

Example Output:
```

Length check: Passed
Capital check: Failed
Number check: Passed
Symbol check: Passed
Digit sum check: Failed
Positivity check: Passed
Palindrome check: Passed

```
